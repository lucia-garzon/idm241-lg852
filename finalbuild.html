Rules

Hover ON

Overall Card

  • Hover animation eases in within 0.3 seconds
    • Card increases in size
    • Card's drop-shadow grows on all sides
    • Hover bar eases up (translateY(0)) within 0.5 seconds, going from 0 to 1 opacity in 0.3 seconds simultaneously

Card Image

  • Hover animation eases within 0.3 seconds
    • Card image increases in size by 10%

"View Case Study" Button

  • Cursor changes to finger immediately
  • Hover animation happens smoothly in 0.3 seconds (using transition: all .3s ease .1s;), starting after a 0.1-second delay
    • Button text changes to bold
    • Button moves up by 5 pixels

Logo Icon(s)

  • Icon scales and rotates 4 degrees smoothly within 0.3 seconds
  • Cursor changes to finger immediately
  • Tooltip appears using smooth easing animation within 0.3 seconds
    • Scales to full size (scale(1)) and transitions to full opacity (opacity: 1)

Hover Bar

  • Hover animation eases within 0.3 seconds with a 0.1-second delay
    • Background color changes to darker color

Hover OFF

Overall Card

  • Card and shadow return to original state with the same 0.3-second animation.
  • Hover bar eases back down (translateY(20px)) within 0.5 seconds, going from 1 to 0 opacity in 0.3 seconds simultaneously

"View Case Study" Button

  • Cursor changes back to pointer immediately
  • Same 0.3-second animation with the 0.1-second delay is applied
    • Button text goes back to its original font weight
    • Button back down to original position

Logo Icon(s)

  • Icon reverses to original state smoothly within 0.3 seconds
  • Cursor changes back to pointer immediately
  • Tooltip eases out smoothly within 0.3 seconds
    • Scales down (scale(0)) and transitions to full transparency (opacity: 0)

Hover Bar

  • Hover animation eases within 0.3 seconds with a 0.1-second delay
    • Background color reverts to original color

Click

"View Case Study" Button

  • The finger reverts back to a cursor
  • Button scales down to 90% of its original size within 0.3 seconds
  • Button opacity reduces to 0.9, fading smoothly within 0.15 seconds
  • The event.preventDefault()method blocks the button's default hyperlink action
  • The clicked class triggers the animation (scaling and fading)
  • After the animation, the class is removed, resetting the button
  • Page navigation is delayed by 0.3s, then the case study page opens in a new tab

Logo Icon(s)

  • Click animation occurs smoothly within 0.4 seconds using ease
    • Icon scales down to 0.85 and rotates by 8 degrees (transform: scale(0.85) rotate(8deg))
    • Opacity is reduced to 0.5
  • The function handleIconClick is triggered, and retrieves the data-link and data-type attributes from the clicked icon
  • If the data-type is 'section' or 'external', the page opens in a new tab (via window.open(link, '_blank'))
  • The click animation (scale and opacity change) remains for 600ms before returning to the icon's original state
  • Simultaneously, page navigation is delayed by 600ms, then a new page opens in a new tab (either to a section of a case study page or an external link)
  • The 'clicked-logo' class is removed after the animation ends, restoring the icon to its original state
