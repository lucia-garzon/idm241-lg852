Beta - Lucia Garzon Portfolio

Triggers

"View case study" Button

  • Hover ON
    • 0.1 second delay
  • Hover OFF
    • 0.1 second delay
  • Click

Overall Card

  • Hover ON
  • Hover OFF

Logo Icon

  • Hover ON
  • Hover OFF

Rules

Hover ON "View case study" Button

  • Cursor changes to finger immediately
  • Button text changes to bold
  • Button moves up by 5 pixels
  • The hover animation happens smoothly with a 0.3-second transition (using transition: all .3s ease .1s;), starting after a 0.1-second delay

Hover OFF "View case study" Button

  • Cursor changes back to pointer immediately
  • Button text goes back to its original font weight
  • Button back down to original position
  • Same 0.3-second transition with the 0.1-second delay is applied

Click "View case study" Button

  • The finger reverts back to a cursor
  • The current page disappears, loading a new page (case study page)

Hover ON Overall Card

  • Card scales in size
  • Card's drop-shadow grows on all sides
  • The hover animation eases in within 0.3 seconds

Hover OFF Overall Card

  • Card and shadow return to original state with the same 0.3-second transition.

Hover ON Logo Icon

  • Icon scales and rotates 4 degrees smoothly within 0.3 seconds
  • Cursor changes to finger immediately

Hover OFF Logo Icon

  • Icon reverses to original state smoothly within 0.3 seconds
  • Cursor changes back to pointer immediately
