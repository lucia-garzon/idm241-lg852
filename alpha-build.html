Trigger

"View case study" Button

  • Hover ON
    • 0.1 second delay
    • Transition lasting 0.3s
  • Hover OFF
    • 0.1 second delay
    • Transition lasting 0.3s
  • Click

Rules

Hover ON

  • Cursor changes to finger immediately
  • Button text changes to bold
  • Button moves up by 5 pixels
  • The hover animation happens smoothly with a 0.3-second transition (using transition: all .3s ease .1s;), starting after a 0.1-second delay

Hover OFF

  • Cursor changes back to pointer immediately
  • Button text goes back to its original font weight
  • Button back down to original position
  • Same 0.3-second transition with the 0.1-second delay is applied

Click

  • The finger reverts back to a cursor
  • The current page disappears, loading a new page (case study page)
